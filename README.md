
# PORTE

* Programmable
* Opinionated
* Repeatable
* Tiny
* Extensible

## Description

PORTE is a minimal programming language providing tools to automate file management like backups, building process or unit testing.

## Usage

> Currently only supported in unix environment.

```sh

$ porte hello-world.pr


```
