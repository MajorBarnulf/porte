
# PROUT

* Programmable
* Repeatable
* Opinionated
* Understandable
* Tiny

## Description

PROUT is a minimal programming language providing tools to automate file management like backups, building process or unit testing.

## Usage

> Currently only supported in unix environment.

```sh

$ prout hello_world.pr


```
